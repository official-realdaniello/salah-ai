Technopark.ps Demo Project

If the live demo is no longer available, it is likely because the free tier expired.
You can host this project by yourself on [Render](https://dashboard.render.com/web/new) by following these steps:

1 - Create a Web Service 

2 - Paste this GitHub repository link https://github.com/official-realdaniello/salah-ai/

3 - Run npm install >(then) node server.js

  - Render will provide you with a temporary live link

To keep the demo online and prevent it from going to sleep, follow these steps:

1 - Use [UptimeRobot](https://dashboard.uptimerobot.com/onboarding/1)

2 - Create an HTTP monitor

3 - Paste your Render link

Demo is available at https://salah-ai.onrender.com/

This project has been uploaded to GitHub for open-source code review. I don't need the source code anymore. The purpose is to demonstrate my ability to build a full project, including features and design, in a short time.
This entire project was completed in less than 12 hours.

[ Info ]:
- The site may give poor results because it’s using free API keys. If you replace them with paid API keys, the outputs (answers/images) will be way better. So bad results don’t mean the site is broken, just that the API tier is limited.
